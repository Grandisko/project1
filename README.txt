# project1

Менеджер заказов и склада.

Приложение имеет возможность добавления, изменения, удаления товаров.
Приложение также может выполнять роль CRM (записывать данные о клиенте и его заказах в базу и выводить на просмотр).    
Приложение ведет подсчет кол-ва товаров на складах (складов может быть несколько).
Приложение умеет автоматически создавать документы типа word + excel для операций(продажа, приемка товара, перемещение, списание товара), при создании документа открывает его при необходимости.
Приложение работает взаимодействуя с базой данных SQLite.

## Клонирование репозитория

Для клонирования репозитория используйте следующую команду:

1. git clone https://github.com/Grandisko/project1

## Переход в директорию проекта

После клонирования репозитория перейдите в директорию проекта:

2. cd <название директории проекта>

## Создание и активация виртуального окружения

Рекомендуется использовать виртуальное окружение для установки зависимостей. Для создания и активации виртуального окружения используйте следующие команды:

3. python -m venv venv

### Windows

4. venv\Scripts\activate

### macOS и Linux

4. source venv/bin/activate

## Установка зависимостей

После активации виртуального окружения установите необходимые зависимости из файла `requirements.txt`:

pip install -r requirements.txt

## Запуск проекта

Теперь вы можете запустить проект. Убедитесь, что вы находитесь в корневой директории проекта и ваше виртуальное окружение активировано.

### Пример команды для запуска

5. python run.py

## Деактивация виртуального окружения

После завершения работы с проектом деактивируйте виртуальное окружение:

6. deactivate
